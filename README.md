A Telegram bot for downloading music from JioSaavn with metadata and album art. Built as a college project.

## Features
- Search songs from JioSaavn
- Download with 128kbps or 320kbps quality
- Full metadata embedding (artist, album, year, cover art)
- Docker containerization with FFmpeg
- Clean Telegram interface

## Prerequisites
- Docker and Docker Compose
- Telegram Bot Token from [@BotFather](https://t.me/botfather)
- Your JioSaavn API (from your repo)
